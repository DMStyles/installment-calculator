# 📱 Multi-Provider Installment Calculator

A premium, dark-themed utility application built using Flutter to help users calculate payment schedules, processing fees, and installment amounts across multiple popular Buy Now Pay Later (BNPL) services in Sri Lanka.

---

## ✨ Features

### 🏠 Smart Dashboard
*   **Activity Overview**: At-a-glance stats — total calculations, comparisons made, and most-used provider.
*   **Recent History**: A unified feed of your last calculations across all three providers.
*   **Quick Actions**: Jump straight into any calculator, comparison tool, or shopping guide.

### 💳 BNPL Calculators
All three providers accessible from one organized hub.

#### 🥥 Koko
*   3-month installment planner with **Find Shop Fee** tab to calculate hidden processing fees.
*   Highlights the default 6% secondary rate charged directly by Koko.

#### ⚡ PayZy
*   Flexible 2, 3, or 4-month payment schedules.
*   Customizable fee percentage (defaults to 8%).

#### 🍃 MintPay
*   3-month installment schedules with adjustable processing fees (defaults to 0%).

---

### ⚖️ Provider Comparison Tool
*   Enter a purchase amount once and compare **total cost, monthly payment, and fees** across all three providers side-by-side.
*   Override fees per provider to reflect real shop rates.
*   Automatically **highlights the cheapest option** for your purchase.

### 🛒 BNPL Shopping Guide
*   Beginner-friendly explanation of how BNPL works.
*   Provider-specific tips and common pitfalls to avoid.
*   Answers to frequently asked questions about BNPL in Sri Lanka.

---

## 🌐 Web Application Access

This repository contains the built distribution of the Installment Calculator web application.

To run locally, serve this directory using any web server:
```bash
# Using Python's built-in server:
python -m http.server 8000
```
Then open `http://localhost:8000` in your web browser.
