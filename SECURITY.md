# Security Policy

## Supported Versions
| Version | Supported |
|---|---|
| 1.0.x | ✅ Yes |

## Reporting a Vulnerability
If you discover a security vulnerability, please **do not** open a public GitHub Issue.

Instead, contact the maintainer directly by opening a private issue or reaching out via GitHub's private vulnerability reporting feature.

Please include:
- A description of the vulnerability
- Steps to reproduce it
- Potential impact

We will respond within **7 days** and aim to release a fix as soon as possible.
