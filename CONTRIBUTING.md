# Contributing Guidelines

Thank you for your interest in contributing to the Installment Calculator!

## How to Report a Bug
1. Go to the [Issues](../../issues) tab.
2. Click **New Issue**.
3. Describe the bug clearly: what you did, what you expected, and what actually happened.
4. Include your device model and Android/browser version if possible.

## How to Suggest a Feature
1. Open a new Issue and label it as a **Feature Request**.
2. Describe what you'd like added and why it would be useful to other users.

## Submitting Code Changes
1. Fork this repository.
2. Create a new branch: `git checkout -b feature/your-feature-name`
3. Make your changes and test them.
4. Commit with a clear message: `git commit -m "Add: description of change"`
5. Push and open a Pull Request against the `main` branch.

## Code Style
- This project uses Flutter/Dart. Please follow the [Dart style guide](https://dart.dev/guides/language/effective-dart/style).
- Run `flutter analyze` before submitting to make sure there are no lint errors.
